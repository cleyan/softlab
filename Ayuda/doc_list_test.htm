Lista de Test

Esta es la Lista de Test, en ella podemos editar un test, editar los valores de referencia de este y los insumos que utiliza.

Buscar Test:

Para comenzar podemos seleccionar uno o varios test segun los criterios de busqueda, el primero de estos criterios es el Codigo Fonasa, (el formato de escritura es sin guiones, ej: 0301045 = Hemograma, o parte de este, ej: 0301 para los Hematologicos).

Código Test corresponde a un nombre referencial corto (idealmente menos de 20 caracteres), en Test podemos escribir parte del nombre, o palabras de referencia separadas por un signo % (ej: acido % plasma = Acido Urico Plasma).

El termino Aislado corresponde a exámenes que pueden ser solicitados solos o formando parte de un exámen Compuesto en cambio exámenes que no cumplan con esta calificación no pueden ser solicitados colos (ej: Cilindros de una orina)

El termino Compuesto corresponde a exámenes que se componen de varios otros (ej: Perfil Bioquimico).

Acceso Rápido es una propiedad de los exámenes que les permiten estar en listas especiales prioritarias por ser exámenes recurrentes se priorizan en su visualización dentro de este sistema de gestión.

En la opción registros por pagina se selecciona el nímero de registros que se desplegaran en cada busqueda.

Una ves selecciondos nuestros criterios de busqueda debemos hacer click en Buscar para logras el despliege de esta.

Al comienzo de cada fila encontraremos un icono el que nos permitira desplegar un recuadro con las opciones de editar el Test, editar los valores de referencia de este y editar los insumos que utiliza.
