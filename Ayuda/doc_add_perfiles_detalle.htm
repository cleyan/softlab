Agregar/Editar Perfiles

Esta area de Softlab se utiliza para crear y editar test compuestos (perfiles, exámenes que incluyan varias opciónes, etc).

Nombre del Perfil:

Para comenzar debemos selecciónar un test de la lista desplegable; si el test aun no esta creado, debemos hacer click en Agregar para ingresar el nombre del test que queremos crear, si ya tubiese otros test asignados estos se desplegaran en el area Test Asignados, para quitarlos de esta lista podemos hacer click en [ << ] ó doble click directamente sobre el test en cuestión.

Test:

En esta area se despliega el listado total de test, también se encientra un area de texto (superior) en la que podemos escribir el nombre del test en busqueda o parte de el.

Para agregar uno o varios test al perfil podemos hacer doble click sobre este o hacer click sobre [ >> ].

Finalmente haremos click sobre uno de los botones de confirmación, Grabar, Borrar ó Cancelar.
